# Premier League Simulator ⚽

A realistic Premier League season simulator built with Next.js.

Features:

- Full 38-gameweek simulation
- Chaos mode and realistic mode
- Form-based match engine
- Title race and relegation pressure
- Rare legendary scorelines
- Season boosts for variance

## Tech Stack

- Next.js
- TypeScript
- Tailwind
- Custom match simulation engine

## Run locally

npm install
npm run dev
